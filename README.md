# 起床レーティング
起床時刻に応じてパフォーマンスが計算されレーティングが更新される。
パフォーマンス、レーティングの色はAtCoder準拠。

## Mojave
iconはmojave砂漠。Macのダイナミックディスプレイ準拠。時刻に応じてiconが変化する。本来16枚だが見分け安い7枚を選んだ。(スマホで見るとiconが変化しないバグがある。)
